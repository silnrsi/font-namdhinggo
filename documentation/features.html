Namdhinggo - Font Features

Namdhinggo is a Unicode font for the Limbu writing system of Nepal. It includes a number of optional features that may be useful or required for particular uses. This document lists all the available features.

These OpenType features are specified using four-letter tags (e.g. 'ss18'). For more information on how to access OpenType features in specific environments and applications, see Using Font Features.

This page uses web fonts (WOFF2) to demonstrate font features and should display correctly in all modern browsers. For a more concise example of how to use Namdhinggo as a web font see Namdhinggo Webfont Example. For detailed information see Using SIL Fonts on Web Pages.

If this document is not displaying correctly a PDF version is also provided in the documentation/pdf folder of the release package.
